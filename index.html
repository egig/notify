Usage

Include css and js files to your page first, don't forget to include jquery and bootsrap as well of course.

<link href="css/notify.css" rel="stylesheet">
<script src="js/notify.js"><script>

Quick Usage

Example:
$.notify('Message', 'error');

Syntax

$.notify(message, status);
message is string, your message. Required.
status is string, either one of 'default', 'info', 'warning', 'error' or 'success'.

$.notify(option);
option is object. Undocumented yet.
